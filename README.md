# Notification Manager
Metric 데이터에 대해 Notification(Alert) 발송 및 관리 기능을 제공합니다.

# How to run
- 기동
	- default profile: java -jar thy-notification.jar
	- production profile: java -jar thy-notification.jar --spring.profiles.active=production
- index (http://localhost:8080/) 페이지를 접근하면 테스트 목록 표시

# Config files
- application.yml
- logback.xml
